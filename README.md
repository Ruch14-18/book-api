-> book-api
-> Book REST API (Node.js + Express)

-> Objective
Build a simple REST API to manage books using in-memory storage.

->  Features
- GET all books(get.png)
- POST add book(post.png)
- PUT update book(put.png)
- DELETE remove book(delete.png)
- Preview(finalop.png)

-> How to Run
npm install
node server.js

Server runs on:
http://localhost:3000

-> API Endpoints

GET /books  
POST /books  
PUT /books/:id  
DELETE /books/:id  

-> Output
Screenshots of API responses are included in the screenshots folder.

-> Tools Used
Node.js  
Express  
Postman

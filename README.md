-> book-api
-> Book REST API (Node.js + Express)

-> Objective
Build a simple REST API to manage books using in-memory storage.

->  Features
- GET all books
- POST add book
- PUT update book
- DELETE remove book
- Preview

-> How to Run
npm install
node server.js

Server runs on:
http://localhost:3000

-> API Endpoints

GET /books  
POST /books  
PUT /books/:id  
DELETE /books/:id
